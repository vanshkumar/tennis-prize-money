# Data Files

This directory holds the retained dataset.

- `static/seedDatasetMetadata.json` contains dataset metadata.
- `raw/source-metadata/grandSlam2025Sources.json` contains source metadata.
- `normalized/grandSlam2025MensSingles.json` contains normalized tournament economics records.

The normalized filename is historical. It now includes tournament-total competition-prize rows, total-player-compensation context rows, and Wimbledon financial denominator rows.

Read `../docs/DATA_MODEL.md`, `../docs/DATA_SOURCES.md`, and `../docs/DATA_CAVEATS.md` before changing these files.
