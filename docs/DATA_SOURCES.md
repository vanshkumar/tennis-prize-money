# Data Sources

## Current Scope

Version `0.1.0` shipped a small seed dataset for 2025 Grand Slam men's singles prize money. The current unreleased dataset adds schema-version-2 numerator semantics, Australian Open 2017-2026 competition-prize rows, Wimbledon ratio-ready normal-edition rows for 2017-2025, US Open operating-revenue denominators for 2017-2024 with strict competition-prize ratios where support can be separated, and Roland-Garros clean competition-prize/revenue rows for 2017-2020 and 2022-2025. Roland-Garros 2017-2019 also include FFT analytical tournament surplus. Wimbledon 2020 is treated as canceled; US Open 2019 and Roland-Garros 2021 preserve financial data while leaving unresolved strict numerators unavailable. The active dataset is stored in:

- `data/static/seedDatasetMetadata.json`
- `data/raw/source-metadata/grandSlam2025Sources.json`
- `data/normalized/grandSlam2025MensSingles.json`

The seed rows below remain manually normalized from the cited sources. There is no dashboard import path, refresh pipeline, or source adapter in the current repository shape.
